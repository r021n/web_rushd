# SMA Unggulan Rushd - Website Clone (Top Section)

This project is a clone of the top section (header and hero section) of the official SMA Unggulan Rushd website ([https://rushd.sch.id/](https://rushd.sch.id/)).

The purpose of this project is to practice front-end web development skills using HTML and CSS, focusing on replicating the layout and visual style of the main page's top section from the Rushd website.

## Technologies Used

- HTML5
- CSS3
